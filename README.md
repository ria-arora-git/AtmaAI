# AtmaAI

A retrieval-augmented AI that answers life questions using the Bhagavad Gita and Hindu scriptures.

## Features
- Scripture-grounded responses (RAG)
- No hallucinated advice
- Hindi & English support
- FastAPI backend

## Tech Stack
- Python
- FastAPI
- FAISS
- Sentence Transformers

## Disclaimer
This project provides spiritual interpretations based on scriptures and does not replace professional advice.
